---
description: Show your application tracker — a text summary of counts by status and work-auth flag by default, or launch the optional web dashboard with `dashboard`.
---

Run the `job-track` skill. By default, print a text summary of my tracker (counts
by status and by work-authorisation flag). If I say `dashboard`, launch the
optional Flask dashboard so I can browse each role, its ATS score, and download
the pack files.

To see how those applications are converting (the outcome funnel, breakdowns,
and coaching takeaways), use `/job-analytics`. To record an outcome, use
`/job-analytics log <company> / <role> / <status>`.
